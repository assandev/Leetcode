# LeetCode Performance Prep

This repository tracks my preparation for technical coding assessments (LeetCode-style, startup-level).

The goal is not just solving problems, but:

- Improve pattern recognition speed
- Optimize time complexity instinctively
- Reduce implementation errors
- Strengthen structured thinking under pressure

All problem statements belong to LeetCode.  
This repository only contains my personal solutions, notes, and performance reflections.
